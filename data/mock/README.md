# Мок-данные для разработки

Эта папка содержит синтетические данные для разработки и тестирования
без необходимости использования реальных .fcs файлов.

## Структура

- `mock_fcs_data.json` — симулированные данные в JSON формате
- `generate_mock_data.py` — скрипт для генерации мок-данных

## Использование

```python
from data.mock.generate_mock_data import MockFCSGenerator

generator = MockFCSGenerator()
data = generator.generate_sample(n_events=10000)
```

## Структура данных

Мок-данные имитируют структуру .fcs файлов:

| Канал | Описание | Диапазон |
|-------|----------|----------|
| FSC-A | Forward Scatter Area | 0 - 262144 |
| FSC-H | Forward Scatter Height | 0 - 262144 |
| SSC-A | Side Scatter Area | 0 - 262144 |
| CD34-APC | Стволовые клетки | 0 - 262144 |
| CD14-PE | Моноциты | 0 - 262144 |
| CD68-FITC | Макрофаги | 0 - 262144 |
| Annexin-V-Pacific Blue | Апоптоз | 0 - 262144 |

## Популяции клеток

Генератор создаёт реалистичные популяции:

- **Живые клетки** (~70%) — FSC/SSC в нормальном диапазоне
- **Debris** (~20%) — низкий FSC/SSC
- **CD34+ стволовые** (~5%) — высокий CD34
- **Макрофаги** (~3%) — высокий CD14/CD68
- **Апоптотические** (~2%) — высокий Annexin-V
