## Фаза 2: Математическое ядро (Неделя 5-8)

### 2.1 SDE модель (глобальный уровень)
- [ ] Реализация уравнения Ланжевена:
  ```
  dNₜ = [rNₜ(1 - Nₜ/K) + αf(PRPₜ) + βg(PEMFₜ)]dt + σNₜdWₜ
  ```
- [ ] Функции воздействия:
  - f(PRP) = C₀e^(-λt) — экспоненциальное затухание
  - g(PEMF) = 1/(1 + e^(-k(f-f₀))) — сигмоидальный отклик
- [ ] Численное решение: метод Эйлера-Маруямы
- [ ] Граничные условия (отражающая граница Nₜ ≥ 0)

### 2.2 ABM модель (локальный уровень)
- [ ] 2D сетка (50x50 для MVP)
- [ ] Типы агентов:
  - Стволовые клетки (пролиферация, дифференциация)
  - Макрофаги (воспаление, очистка)
  - Фибробласты (ремоделирование)
- [ ] Правила агентов:
  - Пролиферация (зависит от PRP)
  - Миграция (хемотаксис к факторам роста)
  - Апоптоз (стохастический + воспаление)
  - Взаимодействие (соседние клетки)

### 2.3 Интеграция SDE + ABM
- [ ] Синхронизация временных шагов
- [ ] ODE → глобальные цитокины → ABM
- [ ] ABM → средняя плотность → ODE
- [ ] PEMF влияние на сетку

### 2.4 Параметры модели
| Параметр | Описание | Диапазон | Источник |
|----------|----------|----------|----------|
| r | Скорость пролиферации | 0.1-0.5 day⁻¹ | Литература |
| K | Ёмкость среды | 10⁶-10⁷ cells/cm² | Калибровка |
| α | Коэффициент PRP | 0.1-1.0 | Калибровка |
| β | Коэффициент PEMF | 0.01-0.1 | Калибровка |
| σ | Волатильность | 0.01-0.1 | Калибровка |
| λ | Затухание PRP | 0.1-0.5 day⁻¹ | Литература |
| f₀ | Оптимальная частота PEMF | 50-75 Гц | Литература |

## Метрики успеха MVP

1. **Функциональность**
   - [ ] Загрузка .fcs файлов работает
   - [ ] Симуляция запускается и завершается < 5 минут
   - [ ] Результаты визуализируются корректно

2. **Качество**
   - [ ] Покрытие тестами > 70%
   - [ ] Нет критических багов

3. **Usability**
   - [ ] Пользователь может пройти workflow за < 10 минут
   - [ ] Документация понятна

4. **Валидация**
   - [ ] Кривые роста соответствуют литературным данным (±30%)
