### Задача 2.2: Реализация SDE модели

**Время:** 20 часов  
**Исполнитель:** Lead Developer  
**Зависимости:** 2.1

**Подзадачи:**

1. **Реализация базового SDE решателя**
   - Метод Эйлера-Маруямы
   - Генерация винеровского процесса
   - Адаптивный шаг времени (опционально)
   - Время: 5 часов

2. **Реализация функции роста**
   - Логистический рост: rN(1 - N/K)
   - Обработка граничных условий
   - Время: 2 часа

3. **Реализация функции PRP воздействия**
   - f(PRP) = C₀e^(-λt)
   - Зависимость от начальной концентрации
   - Учёт разных факторов роста (PDGF, VEGF, TGF-β)
   - Время: 3 часа

4. **Реализация функции PEMF воздействия**
   - g(PEMF) = 1/(1 + e^(-k(f-f₀)))
   - Сигмоидальный отклик
   - Параметры: частота, интенсивность
   - Время: 3 часа

5. **Интеграция компонентов**
   - Полное уравнение dNₜ
   - Комбинированное воздействие PRP + PEMF
   - Время: 3 часа

6. **Тестирование и валидация**
   - Unit-тесты для каждой функции
   - Сравнение с аналитическими решениями
   - Проверка сходимости
   - Время: 4 часа

- [ ] Тесты проходят
