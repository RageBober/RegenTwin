# Описание: generate_mock_data.py

## Обзор

Модуль для генерации синтетических flow cytometry данных.
Используется для разработки и тестирования без реальных .fcs файлов.

---

## Классы

### MockFCSMetadata

**Назначение:** Хранение метаданных синтетического .fcs файла.

**Атрибуты:**

| Атрибут | Тип | Описание | Значение по умолчанию |
|---------|-----|----------|----------------------|
| filename | str | Имя файла | "mock_sample.fcs" |
| n_events | int | Количество событий | 10000 |
| n_channels | int | Количество каналов | 7 |
| cytometer | str | Название цитометра | "MockCytometer v1.0" |
| date | str | Дата создания | Текущая дата |
| channels | tuple[str] | Названия каналов | FSC-A, FSC-H, SSC-A, CD34-APC, CD14-PE, CD68-FITC, Annexin-V-Pacific Blue |

---

### MockFCSGenerator

**Назначение:** Генератор синтетических данных flow cytometry.

**Конструктор:**

```python
def __init__(self, seed: int | None = 42) -> None
```

| Параметр | Тип | Описание |
|----------|-----|----------|
| seed | int \| None | Seed для RNG, обеспечивает воспроизводимость |

---

## Методы

### generate_sample

**Назначение:** Генерация полного синтетического образца.

**Сигнатура:**

```python
def generate_sample(
    self,
    n_events: int = 10000,
    live_fraction: float = 0.70,
    debris_fraction: float = 0.20,
    stem_fraction: float = 0.05,
    macrophage_fraction: float = 0.03,
    apoptotic_fraction: float = 0.02,
) -> pd.DataFrame
```

**Параметры:**

| Параметр | Тип | Описание | По умолчанию |
|----------|-----|----------|--------------|
| n_events | int | Общее количество событий | 10000 |
| live_fraction | float | Доля живых клеток | 0.70 |
| debris_fraction | float | Доля debris | 0.20 |
| stem_fraction | float | Доля стволовых клеток | 0.05 |
| macrophage_fraction | float | Доля макрофагов | 0.03 |
| apoptotic_fraction | float | Доля апоптотических | 0.02 |

**Возвращает:** `pd.DataFrame` с колонками для каждого канала.

**Алгоритм:**

1. Вычислить количество клеток каждого типа
2. Сгенерировать каждую популяцию отдельно
3. Объединить в один DataFrame
4. Перемешать строки для реализма

---

### _generate_live_cells

**Назначение:** Генерация живых клеток с нормальными характеристиками.

**Параметры распределения:**

| Канал | Среднее | Std | Описание |
|-------|---------|-----|----------|
| FSC-A | 100000 | 20000 | Нормальный размер |
| FSC-H | 90000 | 18000 | Нормальная высота |
| SSC-A | 50000 | 15000 | Средняя гранулярность |
| CD34 | 5000 | 2000 | Низкий (негативные) |
| CD14 | 8000 | 3000 | Низкий |
| CD68 | 3000 | 1500 | Низкий |
| Annexin-V | 2000 | 1000 | Низкий (живые) |

---

### _generate_debris

**Назначение:** Генерация debris (мусора, обломков клеток).

**Характеристики:**
- Низкий FSC-A (< 30000)
- Низкий SSC-A (< 20000)
- Случайные значения маркеров

---

### _generate_stem_cells

**Назначение:** Генерация CD34+ стволовых клеток.

**Характеристики:**
- Нормальный FSC/SSC
- Высокий CD34 (> 100000)
- Низкие другие маркеры
- Низкий Annexin-V

---

### _generate_macrophages

**Назначение:** Генерация макрофагов (CD14+/CD68+).

**Характеристики:**
- Высокий SSC (гранулярные)
- Высокий CD14 (> 80000)
- Высокий CD68 (> 60000)
- Вариабельный Annexin-V

---

### _generate_apoptotic

**Назначение:** Генерация апоптотических клеток.

**Характеристики:**
- Сниженный FSC (клетки сжимаются)
- Высокий Annexin-V (> 100000)
- Вариабельные маркеры

---

### to_dict

**Назначение:** Конвертация DataFrame в словарь для сериализации.

**Возвращает:**

```python
{
    "metadata": {...},
    "channels": ["FSC-A", ...],
    "data": [[...], ...]  # Список списков значений
}
```

---

### save_json

**Назначение:** Сохранение данных в JSON файл.

**Параметры:**

| Параметр | Тип | Описание |
|----------|-----|----------|
| df | pd.DataFrame | Данные для сохранения |
| filepath | str | Путь к файлу |

---

## Функции

### create_mock_dataset

**Назначение:** Создание полного набора мок-данных.

**Создаёт файлы:**
- `mock_normal.json` — нормальный образец
- `mock_inflamed.json` — образец с воспалением (больше макрофагов)
- `mock_regenerating.json` — регенерирующий образец (больше стволовых)

---

## Примеры использования

```python
# Создание генератора
generator = MockFCSGenerator(seed=42)

# Генерация образца
df = generator.generate_sample(n_events=10000)

# Сохранение в JSON
generator.save_json(df, "data/mock/sample.json")

# Создание полного набора данных
create_mock_dataset("data/mock")
```
