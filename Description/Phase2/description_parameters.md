# parameters.py — Полный набор параметров модели регенерации

## Назначение

Единый dataclass `ParameterSet` с 90+ параметрами из §8 RegenTwin Mathematical Framework.
Используется всеми модулями расширенной SDE системы: `extended_sde.py`, `wound_phases.py`.

Группы параметров:
- Клеточные: пролиферация, смерть, переключение, carrying capacity
- Цитокиновые: секреция, деградация, ингибирование
- ECM: продукция коллагена, MMP, фибрин
- Шум: sigma для всех 20 переменных
- Вспомогательные: damage signal, кислород
- Численные: dt, t_max, epsilon

Подробное описание: Description/Phase2/description_parameters.md

### ParameterSet.validate

**Назначение:** Проверка физической осмысленности всех параметров.

**Сигнатура:**

```python
def validate(self) -> bool
```

**Поведение:**
1. Проверить все скорости (r_*, delta_*, gamma_*, k_*) > 0
2. Проверить все carrying capacity (K_F, K_E, K_S) > 0
3. Проверить все sigma >= 0
4. Проверить все константы полунасыщения > 0
5. Проверить dt > 0, t_max > 0, epsilon > 0
6. Вернуть True если всё валидно

**Тестовые сценарии:**

| Сценарий | Ожидание |
|----------|----------|
| ParameterSet() (defaults) | validate() == True |
| r_F = -0.01 | ValueError |
| K_F = 0 | ValueError |
| sigma_P = -1 | ValueError |
| dt = 0 | ValueError |
| gamma_TNF = 0 | ValueError |

**Edge cases:**
- Все нули → множественные ValueError
- Очень большие значения (1e15) → допустимо (не ограничиваем сверху)
- n_hill = 0 → ValueError (бессмысленный Hill)

**Инварианты:**
- Если validate() возвращает True, все параметры физически осмысленны
- ValueError содержит имя нарушенного параметра

### ParameterSet.to_dict

**Назначение:** Сериализация всех параметров в dict.

**Сигнатура:**

```python
def to_dict(self) -> dict[str, float | int]
```

**Поведение:**
1. Собрать все поля dataclass в dict {имя: значение}
2. Вернуть новый dict (не ссылку на внутреннее хранилище)

**Тестовые сценарии:**

| Сценарий | Ожидание |
|----------|----------|
| ParameterSet().to_dict() | dict с >= 90 ключами |
| to_dict()["r_F"] | 0.03 |
| to_dict()["n_hill"] | 2 (int) |

**Инварианты:**
- len(result) == количество полей ParameterSet
- Все значения — float или int
- result["r_F"] == self.r_F для любого поля

### ParameterSet.from_dict

**Назначение:** Десериализация из dict.

**Сигнатура:**

```python
@classmethod
def from_dict(cls, data: dict[str, float | int]) -> ParameterSet
```

**Поведение:**
1. Фильтровать неизвестные ключи (игнорировать)
2. Использовать defaults для отсутствующих ключей
3. Создать и вернуть ParameterSet

**Тестовые сценарии:**

| Сценарий | Ожидание |
|----------|----------|
| from_dict({}) | Эквивалентен ParameterSet() |
| from_dict({"r_F": 0.05}) | r_F == 0.05, остальные defaults |
| from_dict({"unknown_key": 1}) | Игнорировать, вернуть defaults |

**Edge cases:**
- Пустой dict → все defaults
- Неизвестные ключи → игнорировать
- Некорректный тип значения → TypeError

**Инварианты:**
- from_dict(ps.to_dict()) == ps (round-trip)

### ParameterSet.get_literature_defaults

**Назначение:** Фабричный метод для литературных значений.

**Сигнатура:**

```python
@classmethod
def get_literature_defaults(cls) -> ParameterSet
```

**Поведение:**
1. Вернуть ParameterSet() (все defaults уже литературные из §8)

**Тестовые сценарии:**

| Сценарий | Ожидание |
|----------|----------|
| get_literature_defaults() | Эквивалентен ParameterSet() |
| get_literature_defaults().r_F | 0.03 |

**Инварианты:**
- get_literature_defaults() == ParameterSet()
